A typical file delivery configuration would be structured like:

<FileTree>

- OUTPUT_PAGESIZE_USECASE_ADMIN0_ADMIN1_YYYYMMDD
  - ADMIN2
    - ADMIN3
        - output1.ext
        - output2.ext
        - output3.ext

</FileTree>

Where ``` output.ext ``` is the original file named according to its specific administrative level, plus file extension.

In practice, for a *health area* reference map (admin level 4) of Mukeba, in *health zone* Tshishimbi, *antenna* Mbuji-Mayi, province Kasaï-Oriental, *country* République Démocratique du Congo:

<FileTree>

- OUTPUT_A2_REFERENCE_RDC_KASAI-ORIENTAL_20250101
  - MBUJI-MAYI
    - ...
    - TSHILENGE/
    {/* - TSHISHIMBI */}
        - **A2_REFERENCE_RDC_KASAI-ORIENTAL_MBUJI-MAYI_TSHISHIMBI_MUKEBA_1_20250101.jpg**
        {/* - A2_REFERENCE_RDC_KASAI-ORIENTAL_MBUJI-MAYI_TSHISHIMBI_MUDIBA_1_20250101.jpg */}
        - ...
- ...
</FileTree>

... and so on.
